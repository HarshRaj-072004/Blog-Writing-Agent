**Agentic AI Blog Writer**

An end-to-end autonomous content generation system built using LangGraph, LangChain, OpenAI, Gemini, Tavily, and Streamlit.

The system goes beyond simple prompt-based content generation by planning before execution, deciding when internet research is required, decomposing work into parallel subtasks, coordinating multiple worker agents, generating supporting visual content, evaluating its own output, and incorporating human feedback through a review loop.

The application is deployed on Render and provides a complete workflow from topic selection to publication-ready blog generation.
